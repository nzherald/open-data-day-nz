Open Data Day, New Zealand

Saturday, February 21st 2015 - 10:00am to 4:00pm

hosted by The New Zealand Herald

Open Data Day is a gathering of citizens in cities around the world to write applications, liberate data, create visualizations and publish analyses using open public data.

Register

What is Open Data Day?

Open Data Day is a global event focused around collaborating on Open Data or Civic Projects.

As an unconference there is no schedule but inpromptu presentations are encouraged.

Bring your laptops and project ideas! More information on how this event is run around the world is available on the international Open Data website opendataday.org

Register See map
 

Who should attend?

Anyone with an interest in Open Data or Open Government. This is an opportunity to collaborate with the community of Open Data users by asking questions, learning new things and sharing knowledge.

Register See map

What should I expect?

As an unconference style event attendees will have the opportunity to give short impromptu talks. Topics could include:

  • Scraping websites
  • Introduction to Government APIs
  • Data analysis tools

WiFi access and lunch will be provided, so make sure you bring your laptop

Why should I attend?

Whether you are a developer, designer, statistician or just generally interested in Open Government, this event will give you the opportunity to share knowledge and experience and find others to collaborate with.

If you are looking for others to collaborate on a particular project with, have technical questions or if you are curious about the types of data available then this event is for you.

When is it?

Open Data Day is February 21st, hosted at the New Zealand Herald Buildings from 10:00am to 4:00pm. The New Zealand Herald is hosting this event in Auckland on it's Albert Street premises, including lunch. Make sure you register.
